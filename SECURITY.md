# Security Policy

## Supported Versions

The following versions of `go-docker-testsuite` receive active security
updates:

| Version  | Supported          |
| -------- | ------------------ |
| latest   | :white_check_mark: |
| < latest | :x:                |

Only the most recent release receives security patches. Users are
encouraged to always upgrade to the latest version.

## Reporting a Vulnerability

If you discover a security vulnerability in `go-docker-testsuite`, please
report it privately — **do not** file a public GitHub issue.

To report a vulnerability, open a draft
**[GitHub Security Advisory](https://github.com/teran/go-docker-testsuite/security/advisories/new)**
in this repository.

Please include the following details in your report:

- A brief description of the issue
- Steps to reproduce (proof of concept or test case)
- Affected versions
- Any suggested fix (if available)

Reports will be acknowledged within **72 hours**. Once the issue is
confirmed, the maintainer will work on a fix and coordinate a release
timeline. After the fix is released, the vulnerability will be publicly
disclosed via a GitHub Security Advisory.

## Disclosure Policy

- The issue will be investigated and a fix prepared under embargo.
- A new patch release will be published as soon as a fix is ready.
- The advisory will be published on GitHub after the release.

## Comments

We appreciate the community's help in keeping `go-docker-testsuite` and
its users safe. Thank you for responsibly disclosing security issues.
